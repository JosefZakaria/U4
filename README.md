# U4

Multithreaded Java Application

This project is a multithreaded Java application demonstrating thread coordination, synchronization techniques, and safe access to shared resources. It follows a clear MVC-inspired structure and integrates a simple GUI for visualizing thread activity.

**Technical Highlights**

Demonstrates understanding of race conditions, critical sections, and thread blocking
Clean separation between controller, model, and view
Robust handling of shared resources
Documentation

No additional setup required.
